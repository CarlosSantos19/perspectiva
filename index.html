Características de los usuarios del proyecto formativo Automatización servicio cliente Burguercomic

Edad y género

A nivel de cliente toda aquella persona que tenga poder adquisitivo. Este proyecto esta enfocado al restaurante BurguerComic, con la finalidad de poderlos ayudar a una automatización del cliente en cuanto al autoservicio como puede ser,(solicitar su mesa, entrada, comida, pago, etc). No importa edad y género

contexto de uso

El software se realizara a traves de una aplicación web con el fin de que el cliente pueda interactuar con la aplicación y pueda mirar el proceso de solicitud, pago y servicio

Necesidades y objetivos

Esta aplicación ayudara a los clientes en este caso a BurguerComic a que pueda enfocar su servicio en servir un buen producto, también a tener más control del tiempo y así podra cumplir con el producto no superior a 20 minutos creando automatización de procesos en el servicio al cliente

Habilidades técnicas para el manejo de sofware

La capacidad de poder navegar en la web, conocimientos básicos de tecnólogia
